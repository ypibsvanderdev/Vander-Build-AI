# AI Bootcamp - Advanced Engineering

The ultimate technical bootcamp for the next generation of engineers. 100% animated, immersive, and built with "tuff" aesthetics.

## Features
- **Total Immersion**: Glassmorphic UI with deep dark mode.
- **100% Animated**: GSAP-powered scroll triggers and motion effects.
- **High-End Design**: Custom cursors, glowing sections, and modern typography.
- **Responsive**: Fully optimized for all screen sizes.

## Tech Stack
- HTML5 / CSS3
- Vanilla JavaScript
- GSAP (GreenSock Animation Platform)
- Google Fonts (Orbitron & Inter)

## Setup
Simply clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/ypibsvanderdev/Ai-Bootcamp-
cd Ai-Bootcamp-
```
